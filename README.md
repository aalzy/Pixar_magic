# VoiceTranscribe

Приложение для транскрибации голоса в текст с экспортом в различные форматы.

## Возможности

- 🎤 Распознавание речи в реальном времени (Web Speech API)
- 📝 Отображение текста с промежуточными результатами
- 📋 Копирование текста в буфер обмена
- 💾 Экспорт в форматах: TXT, DOCX, PDF, ODT
- 🎨 Минималистичный UI/UX дизайн

## Требования

- Браузер с поддержкой Web Speech API (Google Chrome, Edge, Safari)
- Разрешение на использование микрофона
- Подключение к интернету (для загрузки библиотек)

## Использование

1. Откройте `index.html` в браузере
2. Нажмите "Начать запись"
3. Разрешите доступ к микрофону
4. Говорите - текст будет появляться в поле ввода
5. Нажмите "Остановить" когда закончите
6. Экспортируйте результат в нужном формате

## Примечания

- Приложение работает полностью в браузере
- Для корректной работы требуется HTTPS или localhost
- Распознавание зависит от качества микрофона и окружающей среды
